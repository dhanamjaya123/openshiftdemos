
# simple-quarkus 

This is the helm chart for the project. 

## build and install the helm chart
```bash
$ helm package helm -u
$ helm install <chart-name>
$ helm upgrade <chart-name>
```


## To learn more about the release, try:

```bash
$ helm status quarkus-simple
$ helm get all quarkus-simple
```
